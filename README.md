## 課程內容

本教學涵蓋以下主題：

### 基礎篇
- **前言與核心問題** - 了解 NotebookLM 簡報的特性與編輯挑戰
- **四大修改工具詳解** - PDNob、NotebookLM 編輯器、Canva Pro、Lovart

### 進階篇
- **素色背景提示詞技巧** - 從源頭產出易於編輯的簡報
- **YAML 簡報生成標準作業流程** - 內容骨架與視覺皮膚分離的模組化設計
- **DIY 工具製作** - 用 Gemini Canvas 製作 PDF 轉 PPTX 轉換器

### 實戰篇
- **三大實戰應用場景** - 商務報告、學術海報、行銷提案
- **工具選擇指南** - 根據需求快速選擇合適工具
- **常見問題與解決方案**
